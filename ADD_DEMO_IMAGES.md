# How to Add Demo Screenshots

To add your demo screenshots to the website:

1. Save your 3 demo screenshots as:
   - `demo1.png` (first screenshot)
   - `demo2.png` (second screenshot)
   - `demo3.png` (third screenshot)

2. Place them in the `/public/demos/` directory:
   ```bash
   cp /path/to/your/demo1.png /Users/test/Desktop/HiOS/hios-website/public/demos/demo1.png
   cp /path/to/your/demo2.png /Users/test/Desktop/HiOS/hios-website/public/demos/demo2.png
   cp /path/to/your/demo3.png /Users/test/Desktop/HiOS/hios-website/public/demos/demo3.png
   ```

3. The website will automatically display them in the Demo section

The images will be displayed in a 3-column grid with proper aspect ratio (9:16 for phone screenshots).
